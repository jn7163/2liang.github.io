PHP路由技术的原理与实践

发表于 2015-08-26   |   分类于 后端开发   |     |    阅读次数

0x00 路由实现原理

用户通过指定的URL范式对后台进行访问，URL路由处理类进行处理后，转发到逻辑处理类，逻辑处理类将请求结果返回给用户。

约定URL范式和规则

约定一套自己喜欢的，对搜索引擎友好，对用户友好的URL规则

URL处理类(即路由实现的核心)

对用户请求的URL进行解析处理，获取到用户请求的类，方法，以及Query参数等，并将请求转发给逻辑处理类。

逻辑处理类

处理网站的真实业务逻辑。
