Socket模块学习：使用Python socket实现简单的服务器与客户端

发表于 2014-04-12   |   分类于 Python   |     |    阅读次数

在编程语言中，无一例外的都有关于Socket编程方面的知识。Socket编程也就是网络编程。简单的说，就是实现不同电脑间的通信，比如QQ聊天，就是采用了Socket中的TCP/IP协议实现的。这几天，学习了下Python的Socket模块，贴上分享。

在Python中想要实现网络编程，只需要调用socket模块即可，在Python的socket模块中主要有一下几个相关函数：

Socket()         #创建socket对象
Bind()           #绑定IP以及端口
Listen()          #监听服务
Accept()       #获取连接状态
Send()          #发送消息
Recv()          #接收信息
Close()         #关闭socket连接

其实在python中，实现网络编程，也就是socket编程相对C语言等比较简单，下面附上一个C-S示例。

Socket编程：Client端

# -- coding: utf-8 --

import socket
import time

c = socket.socket(socket.AF_INET, socket.SOCK_STREAM) #创建socket对象
c.connect((‘169.254.144.18’, 1234)) #连接指定IP以及端口

while True:
serverStr = c.recv(1024) #接收服务端消息
print “Server say:”, serverStr
clientStr = raw_input(“Client, please input:”)
c.send(clientStr) #向服务端发送消息
time.sleep(2)

c.close() #关闭连接
Socket编程：Server端

# -- coding: utf-8 --

import socket

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM) #创建socket对象
s.bind((“169.254.144.18”, 1234)) #绑定IP以及端口
s.listen(10) #设置可同时监听客户端数量

while True:
conn, addr = s.accept() #接收连接状态
print “Connected with client:”, addr[0]
while True:
serverStr = raw_input(“Server, please input:”)
conn.send(serverStr) #向客户端发送消息
clientStr = conn.recv(1024) #接收客户端消息
print “Client say:”, clientStr

s.close() #关闭连接
 

上面的C-S示例代码只是一个简单的服务器和客户端，只是仅仅实现了服务器与客户端进行对话的功能。其实python在很多方面都非常突出，相比C等语言实现好多功能都要简单的多，想socket编程就很明显。
