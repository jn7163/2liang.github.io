Sae Memcache 使用方法

发表于 2014-04-12   |   分类于 爱撸码   |     |    阅读次数

Memcache是一个高性能的分布式的内存对象缓存系统，通过在内存里维护一个统一的巨大的hash表，它能够用来存储各种格式的数据，包括图像、视频、文件以及数据库检索的结果等。简单的说就是将数据调用到内存中，然后从内存中读取，从而大大提高读取速度。

0x01 SAE Memcache简介

Sae Memcache 是SAE为开发者提供的分布式缓存服务，用来以共享的方式缓存用户的小数据。用户需要先在在线管理平台创建Memcache,然后通过标准的memcache*函数读写Memcache。使用完全等同于PHP中的Memcache模块。

0x02 开通Sae Memcache功能

在使用Sae Memacache之前，需要先在SAE后台去开通SAE的Memcache功能。开通方法如下图：

进入应用后，点击服务管理–>Memcache–>点此初始化MC–>设置Memcache配额–>成功使用。

0x03 使用Sae Memcache

其实Sae Memcache的使用与PHP的Memcache模块基本一致，只是在连接Memcache服务时，两者的使用方法有些许不同。下面给出两者的代码。

<?php
/
//php中连接memcache服务
$mc = new Memcache;
$mc->connect(“127.0.0.1”, 12000);
/
//建立与Sae memcache的链接,初始化链接
$mc = memcache_init();

//保存数据
$mc->set(‘key1’, ‘Unstring is hero!’, 0, ‘60’);
$val = $mc->get(‘key1’);
echo “Get key1 value:”.$val.”<br />”;

//替换数据
$mc->replace(‘key1’, ‘Unstring is hero(replace)!’, 0, ‘60’);
$val2 = $mc->get(‘key1’);
echo “Get key1 value:”.$val2.”<br />”;

//保存数组
$arr = array(‘sae’, ‘bae’, ‘gae’);
$mc->set(‘key2’, $arr, 0, ‘60’);
$val3 = $mc->get(‘key2’);
echo “Get key2 value:”;
print_r($val3);
echo “<br />”;

//删除数据
$mc->delete(‘key1’);
$val = $mc->get(‘key1’);
echo “Get key1 value:”.$val.”<br />”;

//清除所有数据
$mc->flush();
$val2 = $mc->get(‘key2’);
echo “Get key2 value:”;
print_r($key2);
echo “<br />”;

//关闭连接
$mc->close();
?>

0x04 代码分析

第3~6行：

这段代码为PHP使用memcache功能的连接代码，该功能使用前需要确保你的服务器安装memcache模块。安装方法请自行百度。$mc->connect()中第一个参数为服务器IP，第二个参数为memcache的开放端口。

第7~8行：

这段代码为SAE使用memcache功能时的连接函数，加上后，就可以像使用php的memcache那样使用该功能了。

第10~13行：

该段代码为保存临时数据

 mc->set(‘key1’, ‘Unstring is hero!’,  0, ‘60’);

其中第一个参数，key1是用来定位数据；第二个参数即为需要保存的数据内容；第三个参数是一个标记，默认为0或者MEMCACHE_COMPRESSED即可；第四个参数为该缓存数据的有效期，以‘秒’为单位，如果超过这个时间，Memcache服务端将会自动删除该缓存数据，若默认为0，则为永久有效。

0x05 结尾

SAE 的 Web Service 是分布式环境，所以当用户需要共享的缓存某些 key-value 形式的小数据时，就需要用 Memcache 服务，这样可以快速进行数据响应，而且可以减轻后端存储的压力。在某些情况（自行识别，其实我也不知道具体都什么情况下）推荐大家使用该服务。
