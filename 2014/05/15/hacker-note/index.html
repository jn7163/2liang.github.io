记一次黑客丛林之旅--题解

发表于 2014-05-15   |   分类于 爱撸码   |     |    阅读次数

从群里看到的，有大牛发了一个连接，点击进去看了看，是个黑客游戏，一直很喜欢这种玩意，所以就尝试这去玩玩，第一阶段过了，但是第二阶段还没开始做。

游戏链接：http://www.fj543.com/hack/

第一关：

提示语：仅在浏览器端用脚本进行身份验证是不安全的。(The Client-side authentication is not secure)

基本可以推测到，这个页面应该是用JS来验证密码是否正确的，戳进源代码发现：

<script type=”text/javascript”>
function gogogo(){
    var pwd=document.getElementById(“pass”).value;
    if (pwd==”go54321”) {alert(“OK,过关了！”);
            window.location=”./?level=222”;
        } else {
            alert(“Error:密码错误！”);
            document.getElementById(“pass”).focus();
        }
}
</script>

从这段JS代码中可以看到，当pwd表单值为go54321即可过关，那直接输入密码：go54321

第二题：

提示语：这讨厌的脚本，为什么阻止我！(I hate the script,it prevents me)

通过看提示语知道这个题应该也是JS脚本的问题，而且当你在密码框中不输入值时，为提示你密码框不能为空，当你输入值时，又提示你框中不能填写东西，所以判断应该是用JS脚本来判断了表单数据。直接将浏览器的JS脚本禁用，然后点击GO，直接跳转到第三关。

第三题：

您还没有登录，所以看不到本页的秘密。(You must login to see the secret of this level)

提示语：这该死的网页，凭什么说我没有登录？(Damn page! Why you say that I didn’t login?)

New Way：

我知道管理员很懒，密码应该很简单。可是他懒到连登录界面都没做……

一开始看到这个题，我很习惯的想到是不是新建个COOKIE值就行了，然后就打开了COOKIE Manage，刚好看到有个名称为login的cookie，而且他的值是no，我就很欣喜，哈哈，肯定是将no改为yes就可以了，很潇洒的改过cookie，然后刷新页面，等待着进入第四题，可是，悲剧了，页面还是停在那里，我还以为是我的插件有问题，然后就更换各种方法修改cookie，最终依然是不可以。

无奈后，请教大牛，经过大牛点拨，才发现其实也很简单，为何我没有想到呢。

提示上面有说管理员懒到连登陆页面都没做，再知一般登陆都是用POST提交，直接用firfox插件POST提交一个password，页面跳转，给了去第四题的链接。

为何我就木有想到呢？

第四题：

看到题上那出现好多的点和横杠，很明显莫尔斯电码，直接百度莫尔斯电码，对照着找出每个对应的字符，最终解出：IAMOK

第五题：

提示语：用流行的加密算法把密码加密成e40756427fac3320或YmFzZTY0aXNvaw==的形式并不可靠。

看到上面的提示语一看第一种加密MD5，第二种加密Base64，很明显Base64比较好解密，百度Base64解密，随便找个工具，就可以解出。最终密码：base64isok

第六题：

if mstsc+vnc=9290 then password=MSSQL+MySQL+Oracle

看到这句代码，一开始看不懂什么意思，看到他在一起相加成了数字，在看着后面的密码是由三种数据库加起来的，没头绪，就百度了下vnc是什么东西，百度后发现是远程控制软件，在看到mstsc应该是远程桌面连接，记得不清楚了，百度了下果然是，然后想到是不是端口号加在一块，然后就查了下这两种服务的端口号:3306+5901=9290。刚刚好，于是查了下三种数据库的默认端口，加在一块，可得密码：1433+3306+1521=6260。

第七题：

题上给了一个GIF图片，肉眼毛也看不出来，因为只给了一张图，想着肯定是图上有什么家伙，因为是GIF图，直接下载下来，扔到PS中去，逐帧看图，发现有三个大写的单词，以及三个小写的词，通过逐帧查看，发现说的是忽略大写字母，那直接将三个小写的词放上去，即可。

第八题：

吴世昌的弟弟的网名是什么？(What’s the username of WuShichang’s little brother)

提示语：小小社工，过这一关主要靠人脑，电脑只是辅助。

题目在问吴世昌的弟弟的网名，一看就知道是社工方面的问题，因为之前看了下这个官网，官网上面介绍了，站长就是吴世昌，他说了他的弟弟叫吴其昌。通过提示，主要靠人脑，就得想想了，肯定不是直接从网上找到他的网名，然后发现站长吴世昌的网名是fj543，后面的543感觉是名字的谐音，然后吴其昌就可以试试573，组合后就是fj573，输入试试，果然不出所料，成功。密码：fj573

第九题：

提示语：使用IE 5.43版本的浏览器访问?level=9token可以得到令牌。

从提示上看到说是用IE 5.43版本的浏览器访问指定网页就可以得到令牌了，那就直接修改UserAgent，从网上各种找，想着找找IE 5.43浏览器的UserAgent，可是找了好多找不到，最后就直接根据其他版本的IE浏览器构造，可是无论怎么搞都不行，后来请教大牛，大牛说UserAgent直接修改成IE 5.43即可，可是我用浏览器插件修改成IE 5.43还是不行，用Burp，竟然出故障了，浏览器链接不到Burp的代理，就用了下Fillder，但是还是不行，最后实在不行，直接用了大牛的令牌来到第十题了。

第十题：

这个题给了一个压缩包，加密的，大牛给了我密码，我也具体没看密码怎么来的，因为我现在没法从第九题到第十题的链接，没办法。压缩包里面有一个EXE文件，直接运行，生成了一个txt文件，可是无论如何也打不开该文件，用Hxd也打不开，反正是什么操作都不能用。大牛说这应该是由两个txt文件合并起来的，需要用\.\才能打开，但是试了好久不行，最后大牛说前面要加个type命令，然后加上\.\文件路径，回车，token就出现了。

其实不知道第十题的命令到底是什么情况，明天好好问问大牛。

type命令：http://blog.csdn.net/hudashi/article/details/7066190

至此，第一阶段完成了，以为这就完了，谁知道下面还有下一关，才知道这是第一阶段，下一阶段还没搞呢，改天玩玩。
